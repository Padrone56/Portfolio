Rolfes Agri WebApp

I designed and developed a ASP.Net MVC Web App used by Rolfes Agri (Pty) Ltd.

The WebApp allows farmers and staff to log the results from trails, which were done on various corps across South Africa. The App allows for User Authentication, Email notification and cross linked databases. It provides a central exchange of information from users in rural areas.

Timetable Application

I developed a Windows Presentation Foundation (WPF) application that was used by the Student Assistants of IT Residences to generate Timetables for the University of Pretoria.

The information used for the timetables was aquired by making use of scraped HTML data. Timetables generated avoid all possible clashes, by trying every single possible timetable. This process was sped up by the use of hashing when comparing various timetables.

I aim to update this to an interactive WebApp, when I have time.

Map Explorer

I expanded on the work done by Panagiotis Kapros, used to organize information from Fantasy Maps.

His project was developed on an older implementation of ASP.Net MVC, which I converted to the newest version, along with updates to the Leaflet Package and Javascript. I did some CSS updates and also added a map for my own fantasy continent, Glainia. I added a Map Maker (which I'll update when I have time), that offers an intuitive way of making custom maps, making use of the HTML5 library, Konva. The project can be found here

BrainGain Tutors

I developed a Website for a small startup company, run by students at the University of Pretoria. BrainGain tutors aim to help highschool students with an interactive teaching enviroment.

The website facilitates tutor and student signup, assigning a tutor to a student, and applying for help with various subjects.

Misc Applications

While working as Senior Student Technician at the University of Pretoria, I automated various tasks by implementation of C# Windows Forms applications. This was done by making use of Email Sending, HTML scraping, data recording, and various other batch files and scripts.

Cow Glove

I helped design and implement an Electronic Glove that was used to determine how hard a veterinarian was palpating inside of a cow. This was used as part of research for a PhD done at Onderstepoort Veterinary Institute, a satellite campus of University of Pretoria

The glove formed part of research that would help the implementation of interactive education within the classroom.
